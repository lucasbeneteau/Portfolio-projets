# Administrateur Systèmes, Réseaux et Sécurité

Ce dépôt rassemble les projets réalisés durant ma formation.  
Ils couvrent l’administration système, réseau, cybersécurité et cloud.

---

## Systèmes

| Projet | Technologies | Objectif |
| :--- | :--- | :--- |
| Infrastructure AD multi-sites | Windows Server, ADDS, GPO, VPN IPsec | Mise en place d’une architecture Active Directory multi-sites avec RODC et GPO |
| ITSM & gestion de parc | Linux, Apache, MySQL, GLPI | Déploiement d’un outil de ticketing et gestion du parc informatique |
| Architecture 3-tiers | Linux, Apache, PHP, MySQL | Mise en place d’une architecture applicative Web / App / Database |
| Automatisation Active Directory | PowerShell, Ansible | Automatisation de la création de ressources AD, montage des partages réseau et déploiement de logiciels et mises à jour |
| Sauvegarde & restauration | Linux, rsync, cron | Mise en place de stratégies de sauvegarde incrémentale et différentielle et procédure de restauration |
| Supervision infrastructure | Nagios, Rsyslog | Monitoring d’un serveur WordPress et centralisation des logs |
