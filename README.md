# Administrateur Systèmes, Réseaux et Sécurité

Ce dépôt rassemble les projets réalisés durant et après ma formation.  
Ils couvrent l’administration système, réseau, cybersécurité et le cloud.

## Systèmes

| Projet | Technologies | Objectif |
| :--- | :--- | :--- |
| Infrastructure AD multi-sites | Windows Server, ADDS, GPO, VPN IPsec | Architecture Active Directory multi-sites avec RODC et GPO |
| ITSM & gestion de parc | Linux, Apache, MySQL, GLPI | Déploiement d’un outil de ticketing et inventaire du parc |
| Architecture 3-tiers | Linux, Apache, PHP, MySQL | Mise en place d’une architecture Web / App / Database |
| Automatisation Active Directory | PowerShell, Ansible | Automatisation creation ressources AD, montage reseaux, deploiement logiciel/mise à jour |
| Sauvegarde & restauration | Linux, rsync, cron | Mise en place de stratégies de sauvegarde incrémentale et différentielle et restauration |
| Supervision infrastructure | Nagios, Rsyslog | Monitoring d’un serveur WordPress et centralisation des logs |
