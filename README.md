# Administrateur Systèmes, Réseaux et Sécurité

Ce dépôt rassemble les projets réalisés durant et après ma formation.  
Ils couvrent l’administration système, les réseaux, la cybersécurité et le cloud.

## Systèmes

| Projet | Technologies | Objectif |
| :--- | :--- | :--- |
| Infrastructure AD multi-sites | Windows Server, ADDS, VPN IPsec, PfSense | Architecture Active Directory multi-sites avec RODC et GPO |
| Automatisation Active Directory | PowerShell, Ansible | Automatisation de la création d’OU, utilisateurs et groupes |
| Supervision infrastructure | Nagios, Rsyslog | Monitoring d’un serveur WordPress et centralisation des logs |
| Sauvegarde & restauration | Linux, rsync, cron | Mise en place de stratégies de sauvegarde incrémentale et différentielle |
| ITSM & gestion de parc | Linux, Apache, MySQL, GLPI | Déploiement d’un outil de ticketing et inventaire du parc |

## Réseaux

| Projet | Technologies | Objectif |
| :--- | :--- | :--- |
| Architecture 3-tiers | Linux, Apache, PHP, MySQL | Mise en place d’une architecture Web / App / Database |
| Simulation réseau Cisco | Packet Tracer, VLAN, OSPF, HSRP, ACL, LACP | Conception et configuration d’un réseau d’entreprise |

## Cybersécurité

| Projet | Technologies | Objectif |
| :--- | :--- | :--- |
| Hardening serveur web | Nginx, Apache, Fail2ban, NFTables | Sécurisation des services web et protection contre les attaques |
| Sécurisation SI (ANSSI) | VLAN, Bastion, VPN, SIEM | Conception d’une architecture sécurisée selon les recommandations ANSSI |
| Pentest Active Directory | Kali Linux, BloodHound, Mimikatz | Test d’intrusion interne sur un domaine AD |
